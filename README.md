# Build Server

## Quick Start

create droplet 
drop it and connect 
```bash
git clone https://github.com/howlinbash/compose
cd compose
```

add ENV_VARS to .env 

```bash
docker-compose up -d
```

### Make a change

rewrite .env file with change
```bash
docker-compose up -d
```
